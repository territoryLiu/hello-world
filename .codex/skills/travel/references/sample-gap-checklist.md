# Sample Gap Checklist

Use `sample.html` as the density benchmark. Check whether the current guide has enough detail in these areas:

- 订票顺序
- 逐段交通方案
- 主路线和备选路线
- 中转城市停留方案
- 最近可查班次与价格区间
- 历史天气体感
- 当前月份分层穿衣
- 必备物品
- 每日执行表
- 景点收费与预约时间
- 按城市整理的美食店铺清单
- 每家店的地址、菜系和推荐菜
- 排队与时间安排提示
- 风险与避坑提示
- 值得直接打开的来源链接
