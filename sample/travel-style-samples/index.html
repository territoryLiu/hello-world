Travel Skill Sample Folder

四版旅行攻略风格样机

这里先不改正式产线模板，先把桌面端和移动端的视觉方向做成可直接打开的真实页面。每一版都用同一套旅行内容骨架，便于纯看风格和信息组织方式。

What To Compare

  • 桌面端是否像杂志专题，而不是普通卡片拼盘
  • 移动端是否更适合一屏一屏往下读
  • 路线、穿衣、景点、交通、美食的层级是否清楚
  • 你更愿意把哪一版升级成 travel-skill 默认样式
Style A

黑白杂志风

桌面端走 Editorial / Magazine，移动端保持清爽白卡片和浅灰底。

  • 适合高级、克制、信息密度高的旅行专题
  • 图片与留白占主导，文字像编辑导览
 Style B

目的地画报风

更强调目的地色彩和视觉冲击，像旅行画报或品牌专题页。

  • 适合景点、风景、季节气氛更强的目的地
  • 视觉记忆点最强，首页张力更大
 Style C

轻奢酒店刊物风

更柔和、舒适，像精品酒店房内旅行手册或生活方式刊物。

  • 适合住宿、美食、穿衣和慢节奏攻略
  • 整体最温和，长时间阅读压力小
 Style D

现代信息设计风

更强调交通、价格和多方案对照，移动端是标准 Bento Box 思路。

  • 适合复杂路线、多中转、多方案旅行
  • 读者最快能看懂路线和执行逻辑
